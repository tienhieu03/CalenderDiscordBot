Nếu bạn muốn reload toàn bộ bot bằng cách restart quá trình Python, thì cách tốt nhất là dùng os.execv(). Dưới đây là cách làm chi tiết:

1️⃣ Cập nhật file chính bot.py
Thêm lệnh b!restart để khởi động lại bot:

import os
import sys
import discord
from discord.ext import commands

intents = discord.Intents.default()
bot = commands.Bot(command_prefix="b!", intents=intents)

@bot.event
async def on_ready():
    print(f"✅ Bot {bot.user} đã sẵn sàng!")

@bot.command()
@commands.is_owner()  # Chỉ cho phép chủ sở hữu bot sử dụng lệnh này
async def restart(ctx):
    """Khởi động lại bot"""
    await ctx.send("🔄 Đang khởi động lại bot...")
    os.execv(sys.executable, ['python'] + sys.argv)

bot.run("YOUR_BOT_TOKEN")
📌 Cách sử dụng:
Gõ lệnh trên Discord:

b!restart
👉 Bot sẽ tự động tắt và chạy lại ngay lập tức.

2️⃣ Lưu Ý Khi Dùng os.execv()
Cách này chỉ hoạt động nếu bot chạy từ một terminal hoặc script.
Nếu bot chạy bằng PM2, Docker, hoặc Systemd, bạn có thể cần cách khác (tôi có thể hướng dẫn thêm nếu bạn cần).
3️⃣ Nếu Dùng Bot Trong Môi Trường Ảo (venv)
Nếu bot chạy trong một môi trường ảo (venv), bạn cần thay sys.executable bằng đường dẫn tới Python của venv:

os.execv("/path/to/venv/bin/python", ["python"] + sys.argv)
📌 Bạn có thể kiểm tra đường dẫn Python trong venv bằng lệnh:

which python
